### Sobre

O projeto é uma página web com elementos das Hq's Marvel, a fim de colocar em prática a utilização de linguagens web.

___
### Tecnologias utilizadas

O projeto foi desenvolvido com as seguintes tecnologias

- HTML 
- CSS  
- JAVASCRIPT 
___
### Como baixar o projeto

```bash
#clonar o repositório
$ git clone https://github.com/luanadepaulla/testeMarvel

#entrar no diretório
$ cd testeMarvel

#instalar as dependênicas
$ yarn install

#Iniciar o projeto
$ yarn start
